encodeHuff :: HuffTree Byte -> [Bool] #

Function which reads input.If it comes across a True, then it can be first True (either at the beginning of the code or after a False) then the next bit is part of the tree (represented inversely).Otherwise, it is considered as finalized the code. If a False is read, then the next 8 bits, correspond to a symbol.

The function changes a byte list into another byte list , compressing it The last True is added in order to have an end of file character (All of the False found before the last True are ignored by the decoder)
